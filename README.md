# Bag of Bread

> [!TIP]
> Crumbs and nuggets for the Hovis-inclined

---

## 1. About This Project

A static website built with Svelte 5 and SvelteKit, featuring a playful bread and toast themed design. This site serves as a resource collection with a cartoonish and irreverent aesthetic.

---

## 2. Tech Stack

- **Framework:** Svelte 5
- **Build Tool:** SvelteKit with Vite
- **Adapter:** @sveltejs/adapter-static
- **Hosting:** GitHub Pages
- **Styling:** Custom CSS with bread/toast color palette

---

## 3. Development

### Prerequisites
- Node.js 20+
- npm

### Getting Started

1. Clone the repository
```bash
git clone https://github.com/JasonWarrenUK/Bag-of-Bread.git
cd Bag-of-Bread
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The static site will be generated in the `build/` directory.

### Preview Production Build

```bash
npm run preview
```

---

## 4. Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch via GitHub Actions.

The deployed site will be available at: `https://jasonwarrenuk.github.io/Bag-of-Bread/`

---

## 5. Design Theme

**Color Palette:**
- Background: Wheat and Sandy Brown gradients (#f5deb3, #f4a460)
- Headers: Chocolate and Saddle Brown (#d2691e, #8b4513)
- Accents: Cornsilk and Sienna (#fff8dc, #a0522d)

**Typography:**
- Font Family: Comic Sans MS (for that cartoonish feel)

**Aesthetic:**
- Playful and irreverent
- Bread and toast themed throughout
- Cartoon-inspired visual elements
